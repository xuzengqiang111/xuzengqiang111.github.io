康复心得四：私立康复机构如何选择

发表于 2023-09-15 更新于 2023-09-26
本文字数： 888 阅读时长 ≈ 1 分钟

书接上文，到底如何在选择康复机构时最大限度避免之前提到的那些坑呢？以下我会分享一些我平时会做的事情和评估方法供大家参考。

三个搜索，一个“面试” （按操作流程排序）

搜康复机构类型

最常见的使用康复两字的机构包括：1足疗按摩，2健身房，3私教工作室，4综合诊所（大众点评搜索可见），我个人更偏向于3和4。行业中做的较好的康复师很多都开设了工作室，而综合诊所的从业人员大多都有医疗背书。如果你的问题医疗性更强保健性较弱，这两类机构当中有医疗背景的康复师可能会更适合你。

搜康复师背景

一些康复机构的康复师评级和Tony的机构是一样的，高级康复师和艺术总监没什么区别，都是营销用语，真正区分他们的是硬件背景。比如工作年限，擅长领域，行业职业证书，海外项目参与度等。既然是营销，有任何过硬的背景机构一定会贴出来大肆宣传，简单对比两个机构康复师的背景谁更有优势一目了然。

搜负面评价

想办法找负面评价，门槛最低的就是我们的大众点评，有人会说个人情况不同，其他人有问题不代表我也会有问题，那么负面评价看什么，看的就是出现问题的类型，以及机构的处理方法，是故意差评还是有理有据，其实也比较好辨别，如果商家一味强调自己的合理性而不主动解决问题或干脆忽视，慎选。从别人身上发生的事情里就可以预料到如果事情发生在自己身上，大概会是什么样的结果，这样的结果你能接受吗？你有应对方案吗？

“面试”（面谈+ 试课）

面谈

面谈主要谈的是康复理念，对方对你病情的看法，和能够提供的服务，对方是激进派还是保守派，提出的康复方案和能提供的服务是否适合你目前的身体情况，能否满足你的康复需求？不要怕花一点时间去和对方交流，如果前期都无法清楚沟通，后期出问题的可能性会明显增加。

试课

试课试的是康复师判断是否准确，对方的工作态度，和机构管理水平，比如你在一次课程之后情况变好还是变差了、对方在工作当中是否专心、如何应对你的反馈、是否让你有不安全感、不同工作人员的衔接是否顺畅等。

在3次课之内你都可以当成是试课，如果你能做到3次课内不续费，只是用一次花一次，就会大大减少你为坑买单的可能性。到这里你在选择康复机构时需要避的坑才算告一段落。

康复心得三：私立康复机构的利与弊

发表于 2023-09-07
本文字数： 594 阅读时长 ≈ 1 分钟

一个好的私立康复机构可以为你承接公立医院所不能满足的那些需求，在私利机构你的治疗会变成患者主导，这就意味着患者变成甲方，治疗方式会变得更加灵活，个性化，对方的服务意识会更好，你的选择性也会更多；在私立机构，你不再是一位患者，而是他们想要挽留的一位客户。是不是听起来很不错？

嘿，醒醒嘿，但凡事物过于美好，大多数时间那就真的只是一场美梦～这里不但没那么美好，反而是雷区里的重灾区：鱼龙混杂，缺少行业标准康复理念进入国内时间尚短，目前还没有建立一个广泛认可的从业标准，各种背景的康复师让人眼花缭乱无所适从。

价格和收获不成正比价格虚高，没有规范的行业定价，一线城市从500-2000/小时不等，是公立医院单次价格的几倍，但是收获却靠运气。

推销严重商业性质导致推销充值办卡情况严重，动辄几千上万，付费不再根据医疗效果和需求，而是如何为平均单次治疗省钱，可也许那是你本身就不需要的治疗。

加重损伤风险大康复师资历和病情不匹配或发挥不稳定导致二次损伤屡见不鲜。管理混乱，乱中出错患者的主治康复师给出的治疗方案在机构各层级得不到精准落实，治疗师，理疗师各有所想，却缺少章程的约束，甚至还能出现头痛治脚的情况。

由于这个行业的特殊性，要避开这些大雷，你唯一能做的就是：多做功课。不管是大众点评、熟人推荐、还是KOL提及的康复机构，我全部都踩过雷，所以我建议不管你的消息来源是什么，一定要自己做足功课。

康复心得二：康复科并非十全十美

发表于 2023-09-03
本文字数： 929 阅读时长 ≈ 1 分钟

当骨科和运动医学科大夫没有办法提供更好的治疗后，会被建议去康复科找找其他的治疗方案。本以为找对地方的我还是踩了不少坑，比如理疗方案不匹配，康复师资源不足，不能满足康复需求还造成了二次损伤。

先来聊聊理疗，理疗方面的坑主要分为：
大夫开的理疗不对症根据医院理疗资源和大夫个人习惯，大夫开出的理疗未必都适合你的，有的时候是有什么给你开什么。尤其是未经验证一下开两周的，很多时候作用不大，钱没少花。我建议的解决方案是四个字：“试，停，调，退”。先试一周，不好就停，及时调整，该退就退💰。

个体耐受度差异，作为理疗部常客，我在不同的康复科为多个身体部位做过近10种不同理疗，大多的理疗通过消炎镇痛加速血液循环的方式让你的身体自我修复，但是有一种理疗你要尤其小心：它就是冲击波。冲击波的原理是破坏然后重建，广泛使用在肌腱相关的陈旧性损伤。如果你目前身体机能修复能力不佳（ie.小伤也比普通人修复时间长），且有其他理疗选择，请不要尝试，一个不小心会让你比治疗之前的情况更差。

操作失误很多三甲医院招收实习生来负责理疗，如果对方调了不适合的档位，也会加重病情。我建议提前和医生确定好频率高低，有无热量，在做之前给负责的医务人员提个醒，会好过你做了一半再调节档位或者做完了才发现不对。

康复训练不适用，康复科第二大法宝就是康复训练，和大家的广泛认知不同，知名医院的康复师资源其实非常有限，他们会优先治疗重症患者，和在本院手术过的患者，这就意味着还有大量的中轻度症状的患者没有办法得到有针对性的运动康复。康复科医生给你开出的康复运动指导，动作普适性较强却缺少了针对性，通常是需要做几次才知道是否适合的，可是回家后却无法及时反馈调整。如果你想要尝试，还是那句话，不好就停，不是大夫安排了就一定适合你。还有记得录视频，方便回家跟练。

当然，也不是说公立医院的康复科就完全不能选，如果你正要去看康复科，首先你可以对自己的情况做一个评估:你是急症吗？你可以持续来医院做理疗吗？你可以接受只是缓解症状，而不能彻底解决问题吗？你的自律能力很强吗？你平时身体的自我修复能力强吗？

如果上面的问题你全都答是，那么恭喜你康复科对你来说可能是一个不错的选择，不然康复科能够提供的帮助，可能不足以解决你的问题。

康复心得一：骨科/运动损伤科面诊

发表于 2023-09-02 更新于 2023-10-11
本文字数： 833 阅读时长 ≈ 1 分钟

当我们出现急性损伤的时候最先想到的就是去公立医院，从普通医院到知名大医院，我在郑州到访过的将近有8家。我在公立医院遇到的问题主要包括：挂号难，医生重视度不佳，和治疗方案不匹配。

尤其是知名医院，通常要提前定抢号，有幸抢到，就按照正常流程去看病。见到医生后面临的第一关就是：【如何让大夫耐心听你讲解完你的病情】。为了提高你的表达能力增加效率，我建议你提前整理好自己的病情，包括起因，发病特点，报告等等，尽量用简短的话准确的解释清楚病情，经过整理过的信息方便大夫判断也为你避免了遗漏重要信息的可能。如果大夫心情尚可，并耐心听你讲完病情，那么这一天是幸运的，最起码你知道误诊的可能性大大降低了，如果不能也许需要见缝插针的多传达一些信息方便对方更好的判断病情。

紧接着你马上要面临第二关：【如何在医生了解你不需要手术后，依然得到最适合你的治疗方案】。往往在医生了解到你不需要手术后，你获得的重视度会迅速下降，有的大夫会告诉你回家养着，养养就好了，还有的大夫会给你开几百块钱的药，大多是用不用两可的。这里作为一般人没有办法准确的判断自己的情况和大夫开出的治疗方案是否匹配，大多数人还是买了，求个心安理得万一有用呢。在这种情况下医生的态度非常重要，我建议你可以提前上好大夫了解一下这个大夫的专业态度，从评价和他回答其他人问题的方式上就可以了解。同时，你也可以线上问诊和线下问诊同时进行，多问几个不同的大夫，这样就可以最大限度省去带着伤还要到处跑的时间，和在多余的路费和没用的药费上的开销。

第三关：【如果医生和你说手术不手术都可以】。如果医生这样告诉你，你就要慎重考虑了，手术不是万能的，手术连带的风险也要考虑在内，以及手术也是创伤也需要愈合，这会给后续的康复锻炼增加难度，所以从新考量你的治疗目标是什么是关键，是恢复正常生活吗？是的话很多时候确实是不需要手术的，保守治疗也可以达到，但是如果你是运动员或者酷爱高强度的运动，答案也许会不同。

能做到以上这些，在问诊阶段的躲坑就算是合格了
